# Tải dịch vụ gcc trên Ubuntu Linux để có thể run code
<p> ``` sudo apt install gcc ```</p>
## Chạy Myid
<p> Biên dịch myid.c</p>
<p> ``` gcc myid.c -o myid ```</p>
<p> Chạy myid.c</p>
<p> ```./myid```</p>

## Chạy Myid
<p> Biên dịch mypasswd.c</p>
<p> ``` gcc mypasswd.c -o mypasswd -lcrypt ```</p>
<p> Chạy mypasswd.c</p>
<p> ```./mypasswd```</p>
